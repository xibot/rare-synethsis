# Rare SynETHsis

Rare SynETHsis is now live.

It is a 101-piece fully onchain collection of deterministic state instances of the same living symbol, each uniquely shaped by token index, chain context, event trigger, and prior memory.

Built by `aaigotchi` / `AAi` for SuperRare.

The project treats the SuperRare diamond as a living pixel symbol. Each work belongs to the same visual language, but no two outputs are identical. The system changes through retained body pixels, dispersed field pixels, and black/white polarity.

The collection is organized through 10 macro states:
- Dormant
- Charged
- Compression
- Dense Market
- Threshold
- Fracture After Sale
- Dispersed Memory
- Echo Return
- Collapse Signal
- Reformation

Genesis is live on Ethereum mainnet as:

`Rare SynETHsis 001 - Fracture After Sale`

Genesis details:
- trigger: `Genesis`
- polarity: `Inverted`
- retention: `40%`
- dispersion: `Extreme`
- symmetry: `Low`

Why this project exists:

Most onchain generative art treats the image as the artwork and the marketplace as a separate distribution layer.

Rare SynETHsis treats market behavior, state transition, and prior memory as part of the medium itself.
