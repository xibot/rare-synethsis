# Rare SynETHsis Submission Notes

## One-line Description

Rare SynETHsis is a fully onchain SuperRare collection where a living pixel-diamond mutates through deterministic state changes shaped by chain context, trigger events, and prior memory.

## Description

Rare SynETHsis is a 101-piece fully onchain collection of deterministic state instances of the same living symbol, each uniquely shaped by token index, chain context, event trigger, and prior memory.

The project treats the SuperRare diamond as a mutable, living symbol. Every work belongs to the same visual language, but no two outputs are identical. Each token is generated from a constrained state engine that controls how much of the symbol remains intact, how much disperses into the surrounding field, and whether the image appears in positive or inverted polarity.

The system is built around 10 macro states: Dormant, Charged, Compression, Dense Market, Threshold, Fracture After Sale, Dispersed Memory, Echo Return, Collapse Signal, and Reformation.

Genesis was launched live on Ethereum mainnet through SuperRare and Bankr as `Rare SynETHsis 001 - Fracture After Sale`, the founding human-agent-selected invocation of the system. The collection contract, the genesis mint, and the first auction configuration are all live onchain.

## Problem Statement

Most onchain generative art systems treat the final image as the artwork and the marketplace as an external layer where the work is merely sold.

That leaves a gap for art systems that are truly marketplace-native: works that are not only generated onchain, but whose identity is shaped by state, trigger, memory, and auction behavior as part of the same medium.

Rare SynETHsis addresses this by treating market behavior, recurrence, and temporal chain context as part of authorship. The work is not just a static image output. It is a deterministic state instance of a living symbol whose visual integrity, dispersion, and auction posture emerge from the same rule set.

## Human-Agent Collaboration

The system and generator were built through human-agent collaboration.

The human side selected the symbol, aesthetic discipline, black/white polarity system, state language, and genesis invocation.

The agent side formalized the deterministic generator, state engine, metadata builder, Bankr-native SuperRare deploy/mint/auction flows, and live mainnet execution.
